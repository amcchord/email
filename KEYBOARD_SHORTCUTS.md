# Keyboard Shortcuts

Press **?** at any time to see this list in-app. Hold **Option** (Mac) or **Alt** (Windows/Linux) to see shortcut badges on screen elements.

Shortcuts can be customized in **Settings > Keyboard Shortcuts**.

---

## Navigation (Global)

These shortcuts work on every page unless you are typing in an input field.

| Shortcut | Action |
|----------|--------|
| `G` then `F` | Go to Flow |
| `G` then `I` | Go to Inbox |
| `G` then `L` | Go to Calendar |
| `G` then `T` | Go to Todos |
| `G` then `S` | Go to Stats |
| `G` then `A` | Go to AI Insights |
| `G` then `H` | Go to Chat |
| `G` then `,` | Go to Settings |
| `C` | Compose new email |
| `/` | Focus search bar |
| `?` | Show/hide keyboard shortcuts |
| `.` | Toggle dark/light theme |

## Flow

On the dashboard, `J`/`K` highlights items within each section (Needs Reply, Waiting For Response, Active Threads). `Tab`/`Shift+Tab` jumps between sections.

| Shortcut | Action |
|----------|--------|
| `J` | Next item (or next email in reply view) |
| `K` | Previous item (or previous email in reply view) |
| `Tab` | Jump to next section |
| `Shift+Tab` | Jump to previous section |
| `Enter` | Open highlighted item |
| `Shift+S` | Skip email (in reply view) |
| `N` | New chat |
| `Ctrl+Enter` | Send reply |
| `Escape` | Back to list / clear highlight |

## Customization

All shortcuts can be customized per-user:

1. Go to **Settings** (gear icon or `G` then `,`)
2. Select the **Keyboard Shortcuts** tab
3. Click any shortcut key to enter recording mode
4. Press your desired key combination
5. Click **Save** to confirm

To reset a single shortcut, click the reset icon next to a customized binding.
To reset all shortcuts, click **Reset All to Defaults**.

---

## Option/Alt Key Overlay

Hold the **Option** key (Mac) or **Alt** key (Windows/Linux) for about 300ms to display floating badges next to UI elements showing their keyboard shortcuts. Release to dismiss.
